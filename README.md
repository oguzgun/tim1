# Last War: Survival — Alliance Knowledge Base (v5 FULL + Community Insights)

This package includes a **core wiki** and a **community insights** layer (Reddit/community-style, experience-driven),
with **bidirectional links** between them.

Structure:
- tr/  Turkish
- en/  English

Core Wiki:
- basic/ intermediate/ advanced/
- systems/ (units, buildings, heroes, research, events)
- alliance/ leadership/ governance/
- cheat_sheets/

Community layer:
- community_insights/ (long-form, experience-driven, “why this rule exists”)

Notes:
- Content is written to be server/season-agnostic where names vary.
- All pages use UTF-8 (Turkish chars safe).
